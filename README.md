# MongoDB learning

### Setup

安装 `mongo`

```bash
brew install mongo
```

安装 `brew services`

```bash
brew tap gapple/services
```

启动 `mongo`

```bash
brew services mongo
===> Successfully started `mongodb` (label: homebrew.mxcl.mongodb)
brew services list
Name              Status  User       Plist
mongodb           started *          /Users/*/Library/LaunchAgents/homebrew.mxcl.mongodb.plist
```
